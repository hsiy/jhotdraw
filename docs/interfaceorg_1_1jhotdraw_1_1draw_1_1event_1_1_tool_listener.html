Detailed Description

Interface implemented by observers of org.jhotdraw.draw.tool.Tools.

Design Patterns

Observer
State changes of tools can be observed by other objects. Specifically

DrawingEditor

observes area invalidations of tools and repaints its active drawing view accordingly.
Subject: org.jhotdraw.draw.tool.Tool; Observer: ToolListener; Event: ToolEvent; Concrete Observer: org.jhotdraw.draw.DrawingEditor.
