Detailed Description

An input format implements a strategy for reading a Drawing using a specific format from either an

InputStream

, an

URI

or a

Transferable

.

Typically the format can be recognized by a Mime type or by a file extension. To identify the format used by a file, an appropriate

FileFilter

for a javax.swing.JFileChooser component can be requested from

InputFormat

.

This interface intentionally contains many identical operations like OutputFormat to make it easy, to write classes that implement both interfaces.

Design Patterns

Strategy

InputFormat

encapsulates a strategy for reading drawings from input streams.
