Detailed Description

A floating text area that is used to edit a TextHolderFigure.

Design Patterns

Framework
The text creation and editing tools and the

TextHolderFigure

interface define together the contracts of a smaller framework inside of the JHotDraw framework for structured drawing editors.
Contract: TextHolderFigure, org.jhotdraw.draw.tool.TextCreationTool, org.jhotdraw.draw.tool.TextAreaCreationTool, org.jhotdraw.draw.tool.TextEditingTool, org.jhotdraw.draw.tool.TextAreaEditingTool, FloatingTextField, FloatingTextArea.
