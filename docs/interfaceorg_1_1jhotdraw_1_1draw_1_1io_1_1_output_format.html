Detailed Description

An output format implements a strategy for writing a Drawing using a specific format into an

OutputStream

, an

URI

or a

Transferable

.

Typically a format can be identified by a Mime type or by a file extension. To identify the format used by a file, an appropriate

FileFilter

for a javax.swing.JFileChooser component can be requested from

OutputFormat

.

This interface intentionally contains many identical operations like InputFormat to make it easy, to write classes that implement both interfaces.
