◆ getComponent()

JComponent org.jhotdraw.draw.DelegatorDrawingView.getComponent ( )

Returns the JComponent of the drawing view.

The drawing framework supports only two use cases, where this component may be accessed:

  • The component can be used to add the drawing view to a parent Swing component.
  • The currently active org.jhotdraw.draw.tool.Tool may add child components to the drawing view. The tool must remove these components when it becomes inactive.
