Detailed Description

A line decoration can be used to draw a decoration at the start or end of a line.

Typically a line decoration is set as an attribute value to a org.jhotdraw.draw.BezierFigure using the attribute keys

org.jhotdraw.draw.AttributeKeys.START_DECORATION

and

org.jhotdraw.draw.AttributeKeys.END_DECORATION

.

Design Patterns

Decorator
