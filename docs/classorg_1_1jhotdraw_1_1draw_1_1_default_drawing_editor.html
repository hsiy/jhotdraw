◆ createActionMap()

ActionMap org.jhotdraw.draw.DefaultDrawingEditor.createActionMap ( )
protected

Override this method to create a tool-specific action map, which overrides the action map of the drawing editor.

The implementation of this class creates an action map which maps the following action ID's to the classes which define them:

  • DeleteAction
  • SelectAllAction
  • IncreaseHandleDetailLevelAction
  • MoveConstrainedAction.West, .East, .North, .South
  • MoveAction.West, .East, .North, .South
  • CutAction
  • CopyAction
  • PasteAction
