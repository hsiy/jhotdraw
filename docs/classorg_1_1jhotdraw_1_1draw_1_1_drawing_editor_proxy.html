Detailed Description

DrawingEditorProxy.

Design Patterns

Proxy
To remove the need for null-handling,

AbstractTool

makes use of a proxy for

DrawingEditor

. Subject: DrawingEditor; Proxy: DrawingEditorProxy; Client: org.jhotdraw.draw.tool.AbstractTool.
