Detailed Description

An

UndoableEdit

event which can undo a lossless transform of Figures by applying the inverse of the transform to the figures.

This object is useful for undoing lossless transformations, such as the translation of figures.

If a lossy transforms is performed, such as rotation, scaling or shearing, then undos should be performed with TransformRestoreEdit instead.
