Detailed Description

An input format for importing drawings using one of the image formats supported by javax.imageio.

This class uses the prototype design pattern. A ImageHolderFigure figure is used as a prototype for creating a figure that holds the imported image.

If the drawing is replaced using the loaded image, the size of the drawing is set to match the size of the image using the attributes

AttributeKeys.CANVAS_WIDTH

and

AttributeKeys.CANVAS_HEIGHT

.

Design Patterns

Prototype
The image input format creates new image holder figures by cloning a prototype figure object and assigning an image to it, which was read from data input. That's the reason why

Figure

extends the

Cloneable

interface.
